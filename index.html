Desafio 1 — Login

Reproduza a tela abaixo no App Inventor. Você decide os tamanhos, espaçamentos e cores — desde que fique parecido.

Projeto: Login Aula 1 — Designer
Meu App
👤
Bem-vindo!
Digite seu usuário
Digite sua senha
Entrar
Resultado esperado
Sua missão
Monte esta tela de login no App Inventor. Tudo centralizado, foto de perfil, dois campos de texto e um botão roxo. Tente sem dicas primeiro.
Dicas — clique em "revelar" se travar
1
Como centralizar tudo na tela
Screen1 → Properties → AlignHorizontal: Center · Não esqueça de marcar Scrollable também.
2
Qual componente usa para o campo de senha
Não é o TextBox comum — é o PasswordTextBox. Ele fica em User Interface e esconde o que você digita automaticamente.
3
Como colocar uma foto de perfil
Componente Image → campo Picture → cole esta URL: https://i.pravatar.cc/150
4
Cor roxa exata do botão e do título
Cor do botão e barra: #4A148C · Cor do texto "Bem-vindo!": #4A148C · No Properties, clique na cor → Custom → cole o código.
📌 Nomes que você DEVE guardar para a Aula 2
Label_BemVindo
O label "Bem-vindo!" — vai mudar de texto ao clicar
Btn_Entrar
O botão Entrar — vai receber o evento de clique
⚡ Aula 2 — o que vai acontecer
when Btn_Entrar.Click → set Label_BemVindo.Text to "Login feito!"
Desafio extra: mudar a cor da tela para verde ao clicar.

Resultado esperado
Sua missão
Monte a tela inicial do jogo. Personagem, vidas, pontos e botão JOGAR. Fundo azul céu. Use o personagem que quiser.
Dicas — clique em "revelar" se travar
1
Como colocar o fundo azul na tela
Screen1 → Properties → BackgroundColor → Custom → #5C94FC

Não achei imagem do Mario — o que faço?

#E65100 — Properties → BackgroundColor → Custom

Meu Player

Resultado esperado
Sua missão
Monte o player. O desafio aqui é colocar os 3 botões lado a lado — para isso existe o HorizontalArrangement em Palette → Layout.
Dicas — clique em "revelar" se travar
1
Como colocar os 3 botões lado a lado
Palette → Layout → arraste HorizontalArrangement para a tela → Width: Fill parent → AlignHorizontal: Center → arraste os 3 botões para DENTRO dele.
2
Como fazer o botão ficar redondo
Button → Properties → Shape: oval · Para ficar quadrado igual a um círculo, defina Width e Height com o mesmo valor (ex: 55px).
3
Cor verde do botão Play
#1DB954 — é a cor do Spotify
📌 Nomes que você DEVE guardar para a Aula 2
Label_Musica
O nome da música — vai mudar ao clicar em Próximo
Btn_Play
O botão ▶ — vai mudar de ▶ para ⏸ ao clicar

O botão ⏭ — vai trocar a música
⚡ Aula 2 — o que vai acontecer
when Btn_Play.Click → set Btn_Play.Text to "⏸" when Btn_Proximo.Click → set Label_Musica.Text to "Believer"
⚠️ O segundo clique que volta para ▶ só tem solução com variável — vem na próxima semana!

Desafio 4 — App de Clima

Monte o app de clima. Na Semana 3 este app vai se conectar a uma API real e mostrar a temperatura de verdade de São Paulo!

Projeto: Clima Aula 1 — Designer ⭐ API real na Semana 3
Tempo Agora
🌤
28°C
São Paulo, SP
Parcialmente nublado
Atualizar
Resultado esperado
Sua missão
Monte o app de clima. Parece simples — mas preste atenção nos nomes dos Labels. Eles são essenciais para a Semana 3 quando a API entrar.
Dicas — clique em "revelar" se travar
1
Como fazer o emoji de clima ficar grande
Use um Label — não Image. Text: 🌤 · FontSize: 48. Nomear: Label_Icone para usar na Aula 2.
2
Como deixar a temperatura grande e azul
Label → FontSize: 36 · FontBold: marcar · TextColor: #1565C0. Nomear: Label_Temp.
3
Por que os nomes importam tanto neste app?
Na Semana 3 você vai adicionar o componente Web que busca a temperatura real. O bloco vai dizer: "set Label_Temp.Text to [temperatura da API]". Se o nome estiver errado, o bloco não encontra o componente.
📌 Nomes que você DEVE guardar para a Aula 2 e Semana 3
Label_Icone
O emoji do tempo — vai mudar na Aula 2 e com a API
Label_Temp
A temperatura — vai receber o dado real da API
Btn_Atualizar
O botão — vai disparar a busca na API
⚡ Aula 2 — o que vai acontecer
when Btn_Atualizar.Click → set Label_Temp.Text to "31°C" → set Label_Icone.Text to "☀️"
Semana 3: em vez de "31°C" fixo, vai vir o valor real da internet.
