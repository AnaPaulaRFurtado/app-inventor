# App Inventor

Repositório com aulas práticas utilizando o App Inventor para desenvolvimento de aplicativos mobile.

## Conteúdo
- Introdução à ferramenta
- Criação de interfaces (Designer)
- Programação com blocos (Blocks)
- Desenvolvimento de aplicativos simples

## Objetivo
Ensinar de forma prática os conceitos básicos de desenvolvimento mobile para iniciantes.
